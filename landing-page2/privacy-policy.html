Privacy Policy

This Privacy Policy explains how our extension collects, uses, and shares user data. By using our extension, you agree to the terms outlined below.

Data Collection

We collect the following types of data to provide core functionality:

  • User Settings: Preferences such as username and session timeout, stored locally or synced across devices.
  • Session Data: Cookies and storage data to manage user sessions securely.
  • Feature Requests: User-submitted feature requests and votes sent to our serverless API.

Data Usage

The collected data is used for the following purposes:

  • Personalizing the extension experience based on user preferences.
  • Managing and restoring user sessions securely.
  • Processing feature requests to improve the extension.

Data Sharing

We share user data only with the following parties:

  • Serverless API: Hosted at https://paint-lib-feature-tracker.daostar-one.workers.dev for session and feature request management.

No data is shared with third parties beyond this API.

User Control

Users can manage or delete their data at any time via the extension's settings interface.

Security

We adhere to industry standards for data protection. The extension does not execute remote code and collects only the data necessary for its functionality.

Contact

If you have any questions about this Privacy Policy, please contact us at support@library-ai.com.
